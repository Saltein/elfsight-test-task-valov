# Elfsight Test Task (Rick and Morty API)

Чтобы запустить приложение на локальном хосте нужно:
1. В пустую папку склонировать репозиторий ```git clone https://github.com/Saltein/elfsight-test-task-valov.git```
2. Перейти в появившуюся папку ```cd elfsight-test-task-valov```
3. Установить npm пакеты ```npm install``` или ```yarn```
4. Запустить ```npm run start``` или ```yarn start```
